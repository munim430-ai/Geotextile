# অধ্যায় ২২: পাসপোর্ট আপলোড SOP

## ২২.১ উদ্দেশ্য

শিক্ষার্থীর পাসপোর্ট সঠিক মানে স্ক্যান করে নির্ভুলভাবে পোর্টাল/ড্রাইভে আপলোড করা।

## ২২.২ পাসপোর্ট মানদণ্ড (গ্রহণযোগ্য হতে)

- মেয়াদ পর্যাপ্ত (সাধারণত কমপক্ষে ৬ মাস+; বিস্তারিত Manual 2)।
- ছবি ও তথ্য পরিষ্কার, পঠনযোগ্য।
- চার কোণা দৃশ্যমান, ঝাপসা বা কাটা নয়।

## ২২.৩ ধাপে ধাপে

1. পাসপোর্টের **তথ্য পাতা (bio page)** পরিষ্কার স্ক্যান/ছবি নিন।
2. ফাইল ফরম্যাট: PDF/JPG; রেজল্যুশন ভালো।
3. ফাইল নাম দিন স্ট্যান্ডার্ড অনুযায়ী (Manual 2, File Naming)।
4. পোর্টালের "Passport" সেকশনে আপলোড করুন।
5. Google Drive-এর শিক্ষার্থী ফোল্ডারেও কপি রাখুন (Manual 2)।
6. Remarks/Progress Sheet-এ "Passport Uploaded" নোট করুন।
