# অধ্যায় ২৬: দৈনিক ও সাপ্তাহিক রুটিন

## ২৬.১ উদ্দেশ্য

প্রতিটি কনসালটেন্টের একটি সুশৃঙ্খল দৈনিক ও সাপ্তাহিক কর্মছক নিশ্চিত করা।

## ২৬.২ দৈনিক রুটিন (Daily Routine)

| সময় | কাজ |
|---|---|
| ৯:০০ | উপস্থিতি, সব Sheet খোলা, রাতের লিড কল |
| ৯:৩০–১টা | প্রতি ঘণ্টায় লিড চেক + কল + WhatsApp |
| ১টা–২টা | লাঞ্চ/নামাজ (কভারেজ সমন্বয়) |
| ২টা–৫:৩০ | প্রতি ঘণ্টায় লিড চেক + ফলো-আপ |
| ৫:৩০–৬টা | Remarks/Progress Sheet আপডেট, দিনের রিপোর্ট |

## ২৬.৩ দৈনিক শেষ চেকলিস্ট (End-of-Day)

- [ ] সব নতুন লিড কল হয়েছে (১ ঘণ্টা নিয়ম)
- [ ] প্রতিটি কলের পর WhatsApp
- [ ] সব Remarks হালনাগাদ
- [ ] আগামীকালের ফলো-আপ তালিকা প্রস্তুত
- [ ] দৈনিক সংখ্যা (কল/আগ্রহী/কনভার্সন) রিপোর্ট

## ২৬.৪ সাপ্তাহিক রুটিন (Weekly Routine)

| দিন | ফোকাস |
|---|---|
| সপ্তাহের শুরু | সাপ্তাহিক টার্গেট নির্ধারণ |
| মাঝ-সপ্তাহ | পাইপলাইন রিভিউ, আটকে থাকা লিড |
| সপ্তাহ শেষ | Progress Sheet রিভিউ, সাপ্তাহিক রিপোর্ট ম্যানেজারকে |

**সাপ্তাহিক চেকলিস্ট:**
- [ ] সাপ্তাহিক কল/কনভার্সন সংখ্যা
- [ ] কোল্ড লিড রি-এনগেজমেন্ট
- [ ] Progress Sheet ক্লিনআপ
- [ ] ম্যানেজারকে সাপ্তাহিক রিপোর্ট

## ২৬.৫ সাধারণ ভুল / বেস্ট প্র্যাকটিস

- **ভুল:** ⛔ দিন শেষে আপডেট বাকি রাখা; ⛔ সাপ্তাহিক রিভিউ এড়ানো।
- **বেস্ট প্র্যাকটিস:** ✅ প্রতিদিন একই সময়ে রিপোর্ট; ✅ সপ্তাহের শুরুতেই টার্গেট লিখুন।
