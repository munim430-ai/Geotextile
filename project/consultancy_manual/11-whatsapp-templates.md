## ১১.৩ টেমপ্লেট সংগ্রহ

### T1 — কল ধরার পর (Contacted)
```
আসসালামু আলাইকুম [নাম] ভাই/আপু 🙏
আমি [কনসালটেন্ট নাম], Hangeul Korean Language & Visa থেকে।
এইমাত্র কথা হলো — [প্রোগ্রাম] সম্পর্কে।
আপনার জন্য প্রয়োজনীয় তথ্য:
• প্রোগ্রাম: [KLP/EAP/Bachelor/Master/PhD]
• ইনটেক: [Intake]
• পরবর্তী ধাপ: Application Fee 20,000 BDT (নন-রিফান্ডেবল)
কোনো প্রশ্ন থাকলে জানাবেন। ধন্যবাদ 🇰🇷
```

### T2 — কল ধরেনি (No Answer)
```
আসসালামু আলাইকুম [নাম] ভাই/আপু 🙏
Hangeul Korean Language & Visa থেকে ফোন দিয়েছিলাম, ধরতে পারেননি।
আপনি কোরিয়ায় [প্রোগ্রাম] নিয়ে আগ্রহী ছিলেন।
সুবিধামতো সময়ে জানালে আমরা আবার কল করব 📞
অথবা এখানেই মেসেজে জানাতে পারেন। ধন্যবাদ।
```

### T3 — আগ্রহী, ডকুমেন্ট তথ্য
```
[নাম] ভাই/আপু, আপনার আগ্রহের জন্য ধন্যবাদ 🙏
[প্রোগ্রাম]-এর জন্য প্রাথমিক প্রয়োজন:
• পাসপোর্ট • একাডেমিক সার্টিফিকেট ও ট্রান্সক্রিপ্ট • NID/জন্ম সনদ
বিস্তারিত ডকুমেন্ট তালিকা পরে পাঠাচ্ছি।
Application Fee: 20,000 BDT (নন-রিফান্ডেবল)।
```

```
আপনি এখন আনুষ্ঠানিকভাবে আমাদের একজন শিক্ষার্থী।

```

## ১১.৪ টোন ও ভাষা নিয়ম

- সবসময় সালাম + নাম দিয়ে শুরু।
- ভদ্র "আপনি" সম্বোধন, প্রয়োজনে ভাই/আপু।
- ইমোজি সংযত ও পেশাদার (🙏 🇰🇷 ✅ 🎉)।
- মিথ্যা প্রতিশ্রুতি নয়।
