# অধ্যায় ৩২: সাধারণ রিজেকশন কারণ

## ৩২.১ উদ্দেশ্য
ইউনিভার্সিটি/ভিসা প্রত্যাখ্যানের সবচেয়ে সাধারণ কারণগুলো জেনে আগেভাগে প্রতিরোধ করা।

## ৩২.২ শীর্ষ রিজেকশন কারণ ও প্রতিরোধ
| # | রিজেকশন কারণ | প্রতিরোধ |
|---|---|---|
| 1 | নামের বানান গরমিল (ডকুমেন্টে ভিন্ন) | পাসপোর্ট = মাস্টার রেফারেন্স, সব মেলান |
| 2 | জন্ম তারিখ অসামঞ্জস্য | পাসপোর্ট/NID/জন্ম সনদ ক্রস-চেক |
| 3 | পাসপোর্ট মেয়াদ অপর্যাপ্ত | আগে মেয়াদ যাচাই/রিনিউ |
| 4 | IELTS মেয়াদোত্তীর্ণ/স্কোর কম | ২ বছর ও প্রয়োজন যাচাই |
| 5 | অসম্পূর্ণ ট্রান্সক্রিপ্ট | সব সেমিস্টার নিশ্চিত |
| 6 | দুর্বল/প্লেজিয়ারাইজড SOP | মৌলিক, রিভিউড SOP |
| 7 | আর্থিক প্রমাণ দুর্বল/window dressing | স্বাভাবিক ৬-মাস স্টেটমেন্ট + সলভেন্সি |
| 8 | আয়-ব্যাংক অসামঞ্জস্য | e-TIN/আয় উৎস সঙ্গতি |
| 9 | অনুবাদ/অ্যাটেস্টেশন অনুপস্থিত/ভুল ক্রম | সঠিক ক্রম অনুসরণ |
| 10 | জাল/সম্পাদিত ডকুমেন্ট | শূন্য-জালিয়াতি নীতি |
| 11 | ঝাপসা/অপাঠযোগ্য স্ক্যান | মান যাচাই |
| 12 | ভুল সেকশনে আপলোড | পোর্টাল ভেরিফিকেশন |
| 13 | ডেডলাইন মিস | আগে প্রস্তুতি |
| 14 | ইউনিভার্সিটি-নির্দিষ্ট ফর্ম বাদ | প্রতিটি ইউনিভার্সিটির প্রয়োজন যাচাই |
| 15 | দুর্বল/অস্পষ্ট রিসার্চ প্রপোজাল (PhD) | সুপারভাইজার-সঙ্গতিপূর্ণ প্রপোজাল |

## ৩২.৩ রিজেকশন প্রতিরোধ নীতি
- প্রতিটি ফাইল ১০০-পয়েন্ট চেকলিস্টে পাস।
- Four-Eyes QA বাধ্যতামূলক।
- সন্দেহ হলে সাবমিট নয় — আগে সমাধান।

## ৩২.৪ চেকলিস্ট ও বেস্ট প্র্যাকটিস
- [ ] শীর্ষ ১৫ কারণ ফাইলে অনুপস্থিত
- [ ] প্রতিটি প্রতিরোধ প্রয়োগ
- **বেস্ট প্র্যাকটিস:** ✅ প্রতিটি রিজেকশন হলে কারণ Error Library-তে লগ করুন (অধ্যায় ৩৩)।

## ৩২.৫ এসকালেশন / FAQ / অনুশীলন
- **এসকালেশন:** প্রকৃত রিজেকশন হলে → ম্যানেজার + পুনঃআবেদন পরিকল্পনা।
- **FAQ:** "রিজেকশন হলে ফি ফেরত?" → Application Fee নন-রিফান্ডেবল; তবে সংশোধন/পুনঃচেষ্টা করা হয়।
- **অনুশীলন:** একটি নমুনা ফাইলে শীর্ষ ৫ ঝুঁকি চিহ্নিত করুন।
