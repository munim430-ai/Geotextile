# অধ্যায় ৫: প্রোগ্রাম অনুযায়ী ডকুমেন্ট প্রয়োজনীয়তা

## ৫.১ উদ্দেশ্য

প্রতিটি প্রোগ্রামের (KLP, EAP, Bachelor, Master, PhD) জন্য প্রয়োজনীয় ডকুমেন্টের সুস্পষ্ট তালিকা সরবরাহ করা।

> ℹ️ চূড়ান্ত তালিকা ইউনিভার্সিটি ও ইনটেক অনুযায়ী সামান্য ভিন্ন হতে পারে — সবসময় নির্দিষ্ট ইউনিভার্সিটির প্রয়োজন যাচাই করুন।

## ৫.৩ স্পনসর/আর্থিক ডকুমেন্ট নোট

আর্থিক সক্ষমতা প্রমাণে সাধারণত: ব্যাংক স্টেটমেন্ট + সলভেন্সি + (স্পনসরের) e-TIN/ট্যাক্স + প্রয়োজনে ট্রেড লাইসেন্স/এমপ্লয়মেন্ট + ফ্যামিলি রিলেশন সার্টিফিকেট।
