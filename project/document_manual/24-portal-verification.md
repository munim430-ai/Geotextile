# অধ্যায় ২৪: পোর্টাল ভেরিফিকেশন

## ২৪.১ উদ্দেশ্য
পোর্টালে আপলোডকৃত সব ডকুমেন্ট সঠিক সেকশনে, সঠিক ভার্সনে ও সম্পূর্ণভাবে আছে কিনা যাচাই করা।

## ২৪.২ যা যাচাই করতে হবে
| উপাদান | যাচাই |
|---|---|
| প্রতিটি সেকশন | সঠিক ডকুমেন্ট আপলোডেড |
| ফাইল মান | পঠনযোগ্য, সম্পূর্ণ |
| ভার্সন | সর্বশেষ/সংশোধিত |
| শিক্ষার্থী তথ্য | পোর্টালে নির্ভুল |
| স্ট্যাটাস | প্রতিটি ধাপ সঠিক |

## ২৪.৩ ধাপ
1. পোর্টালে শিক্ষার্থী প্রোফাইল খুলুন।
2. সেকশন-বাই-সেকশন প্রতিটি আপলোড যাচাই করুন।
3. প্রতিটি ফাইল প্রিভিউ দেখে নিশ্চিত হোন (ভুল ফাইল নয়)।
4. অনুপস্থিত/ভুল আপলোড সংশোধন করুন।
5. Progress Sheet-এ "Portal Verified" আপডেট।

## ২৪.৪ Red Flags / সাধারণ রিজেকশন
- ⚠️ ভুল সেকশনে ফাইল।
- ⚠️ পুরোনো ভার্সন রয়ে যাওয়া।
- ⚠️ অনুপস্থিত ডকুমেন্ট।
- ⚠️ ঝাপসা/ভুল ফাইল আপলোড।

## ২৪.৬ এসকালেশন / FAQ / অনুশীলন
- **এসকালেশন:** পোর্টাল এরর/আপলোড ব্যর্থ → অ্যাডমিন + ম্যানেজার।
- **FAQ:** "ভুল ফাইল আপলোড হয়ে গেছে?" → রিপ্লেস/ডিলিট করে সঠিকটি দিন, নোট করুন।
- **অনুশীলন:** একটি প্রোফাইলের সব সেকশন যাচাই করে একটি ভেরিফিকেশন রিপোর্ট লিখুন।
