# অধ্যায় ১৪: ফ্যামিলি রিলেশন সার্টিফিকেট অডিট

## ১৪.১ উদ্দেশ্য
শিক্ষার্থী ও স্পনসরের পারিবারিক সম্পর্ক প্রমাণকারী সনদ (Family Relation/Guardian Certificate) যাচাই করা।

## ১৪.২ যা যাচাই করতে হবে
| উপাদান | যাচাই |
|---|---|
| সম্পর্ক | পিতা/মাতা/অভিভাবক স্পষ্ট উল্লেখ |
| নামসমূহ | পাসপোর্ট/NID-এর সাথে মিল |
| ইস্যুকারী কর্তৃপক্ষ | ইউনিয়ন পরিষদ/পৌরসভা/সিটি করপোরেশন |
| সিল ও স্বাক্ষর | কর্তৃপক্ষের |
| তারিখ | সাম্প্রতিক |
| ইংরেজি ভার্সন | প্রয়োজন হলে অনূদিত |

## ১৪.৩ Red Flags
- ⚠️ নাম NID/পাসপোর্টের সাথে গরমিল।
- ⚠️ কর্তৃপক্ষের সিল/স্বাক্ষর অনুপস্থিত।
- ⚠️ সম্পর্ক অস্পষ্ট।

## ১৪.৪ চেকলিস্ট
- [ ] সম্পর্ক স্পষ্ট
- [ ] নাম ক্রস-চেক
- [ ] কর্তৃপক্ষের সিল/স্বাক্ষর
- [ ] ইংরেজি/অনুবাদ (প্রয়োজনে)

## ১৪.৫ সাধারণ রিজেকশন ও বেস্ট প্র্যাকটিস
- **রিজেকশন:** নাম গরমিল; কর্তৃপক্ষ যাচাই-অযোগ্য।
- **বেস্ট প্র্যাকটিস:** ✅ স্পনসরশিপ ডকুমেন্টের সাথে নাম সঙ্গতি রাখুন।

## ১৪.৬ এসকালেশন / FAQ / অনুশীলন
- **এসকালেশন:** সম্পর্ক প্রমাণে জটিলতা → ম্যানেজার।
- **FAQ:** "স্পনসর নিকট আত্মীয় নয়?" → ইউনিভার্সিটি/ভিসা নীতি যাচাই করুন।
- **অনুশীলন:** একটি সনদে সব নাম NID-এর সাথে মিলিয়ে দেখুন।
