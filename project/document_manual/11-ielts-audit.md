# অধ্যায় ১১: IELTS অডিট

## ১১.১ উদ্দেশ্য
IELTS/ইংরেজি দক্ষতার সনদ যাচাই করে বৈধতা, স্কোর ও মেয়াদ নিশ্চিত করা।

## ১১.২ যা যাচাই করতে হবে
| উপাদান | যাচাই |
|---|---|
| নাম | পাসপোর্টের সাথে মিল |
| TRF নম্বর | উপস্থিত, ভেরিফায়েবল |
| ব্যান্ড স্কোর | Overall + ৪টি মডিউল |
| টেস্ট তারিখ | মেয়াদ ২ বছরের মধ্যে |
| টেস্ট সেন্টার | বৈধ (British Council/IDP) |
| প্রয়োজনীয় স্কোর | ইউনিভার্সিটির চাহিদা পূরণ |

## ১১.৩ Red Flags
- ⚠️ মেয়াদ ২ বছরের বেশি (উত্তীর্ণ)।
- ⚠️ TRF নম্বর অনুপস্থিত/ভেরিফাই হয় না।
- ⚠️ স্কোর ইউনিভার্সিটির প্রয়োজনের নিচে।
- ⚠️ জাল TRF-এর সন্দেহ (কঠোরভাবে যাচাই)।

## ১১.৪ চেকলিস্ট
- [ ] নাম ক্রস-চেক
- [ ] TRF নম্বর ভেরিফাই
- [ ] মেয়াদ বৈধ (< ২ বছর)
- [ ] স্কোর প্রয়োজন পূরণ করে
- [ ] স্ট্যান্ডার্ড নামে সেভ

## ১১.৫ সাধারণ রিজেকশন ও বেস্ট প্র্যাকটিস
- **রিজেকশন:** মেয়াদোত্তীর্ণ; স্কোর কম; জাল সন্দেহ।
- **বেস্ট প্র্যাকটিস:** ✅ অফিসিয়াল ভেরিফিকেশন পোর্টালে TRF যাচাই।

## ১১.৬ এসকালেশন / FAQ / অনুশীলন
- **এসকালেশন:** জাল TRF সন্দেহ → সাথে সাথে ম্যানেজার।
- **FAQ:** "IELTS নেই?" → প্রোগ্রামভেদে বিকল্প (MOI/প্রস্তুতি); ইউনিভার্সিটি নিয়ম যাচাই।
- **অনুশীলন:** একটি TRF-এর মেয়াদ ও ব্যান্ড যাচাই করুন।
