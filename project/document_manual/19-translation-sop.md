# অধ্যায় ১৯: অনুবাদ (Translation) SOP

## ১৯.১ উদ্দেশ্য
বাংলা ডকুমেন্টগুলো নির্ভুল, প্রত্যয়িত ইংরেজি অনুবাদে রূপান্তর নিশ্চিত করা।

## ১৯.২ কোন ডকুমেন্ট অনুবাদ লাগে
- জন্ম সনদ (বাংলা হলে)
- ফ্যামিলি রিলেশন সার্টিফিকেট
- ট্রেড লাইসেন্স
- কিছু একাডেমিক/বোর্ড ডকুমেন্ট (বাংলা হলে)
- অন্য যেকোনো বাংলা সরকারি সনদ

## ১৯.৩ অনুবাদ ধাপ
1. অনুবাদযোগ্য ডকুমেন্ট চিহ্নিত করুন।
2. অনুমোদিত/পেশাদার অনুবাদক দিয়ে অনুবাদ করান।
3. অনুবাদে নাম **পাসপোর্ট অনুযায়ী** নিশ্চিত করুন।
4. অনুবাদকের সিল/স্বাক্ষর/ঘোষণা যুক্ত করুন।
5. মূল + অনুবাদ একসাথে সংরক্ষণ ও ফাইল করুন।
6. প্রয়োজনে নোটারি/অ্যাটেস্টেশনে পাঠান (অধ্যায় ২০–২১)।

## ১৯.৪ অনুবাদ মান যাচাই
| যাচাই | নিশ্চিত করুন |
|---|---|
| নাম/তারিখ/সংখ্যা | মূলের সাথে হুবহু |
| সম্পূর্ণতা | কিছু বাদ পড়েনি |
| অনুবাদক ঘোষণা | সিল ও স্বাক্ষর |
| বানান/গ্রামার | নির্ভুল |

## ১৯.৫ Red Flags / সাধারণ রিজেকশন
- ⚠️ নাম মূল/পাসপোর্টের সাথে ভিন্ন।
- ⚠️ সংখ্যা/তারিখ ভুল অনুবাদ।
- ⚠️ অনুবাদকের প্রত্যয়ন অনুপস্থিত।

## ১৯.৬ চেকলিস্ট ও বেস্ট প্র্যাকটিস
- [ ] সঠিক ডকুমেন্ট অনূদিত
- [ ] নাম/সংখ্যা ক্রস-চেক
- [ ] অনুবাদক সিল/স্বাক্ষর
- [ ] মূল+অনুবাদ একসাথে ফাইল
- **বেস্ট প্র্যাকটিস:** ✅ সবসময় বিশ্বস্ত/অনুমোদিত অনুবাদ সেবা ব্যবহার।

## ১৯.৭ এসকালেশন / FAQ / অনুশীলন
- **এসকালেশন:** অনুবাদ ত্রুটি → পুনঃঅনুবাদ + ম্যানেজার।
- **FAQ:** "নিজে অনুবাদ করা যাবে?" → না; প্রত্যয়িত অনুবাদ প্রয়োজন।
- **অনুশীলন:** একটি অনূদিত জন্ম সনদে নাম ও তারিখ মূলের সাথে মিলিয়ে দেখুন।
