# অধ্যায় ৩৩: রিস্ক ম্যাট্রিক্স ও এরর লাইব্রেরি

## ৩৩.১ উদ্দেশ্য
ডকুমেন্ট প্রক্রিয়ার ঝুঁকিগুলো শ্রেণিবদ্ধ করা এবং অতীত ভুল থেকে শেখার জন্য একটি Error Library বজায় রাখা।

## ৩৩.২ রিস্ক ম্যাট্রিক্স (সম্ভাবনা × প্রভাব)

| ঝুঁকি | সম্ভাবনা | প্রভাব | স্তর | প্রশমন |
|---|---|---|---|---|
| নাম গরমিল | মাঝারি | উচ্চ | 🔴 উচ্চ | মাস্টার রেফারেন্স + QA |
| পাসপোর্ট মেয়াদ | মাঝারি | উচ্চ | 🔴 উচ্চ | আগে যাচাই |
| জাল ডকুমেন্ট | নিম্ন | অতি উচ্চ | 🔴 উচ্চ | শূন্য-জালিয়াতি + যাচাই |
| আর্থিক অসঙ্গতি | মাঝারি | উচ্চ | 🔴 উচ্চ | আয় উৎস ক্রস-চেক |
| অনুবাদ ত্রুটি | মাঝারি | মাঝারি | 🟠 মাঝারি | প্রত্যয়িত অনুবাদ |
| ভুল আপলোড | উচ্চ | মাঝারি | 🟠 মাঝারি | পোর্টাল ভেরিফিকেশন |
| ডেডলাইন মিস | মাঝারি | উচ্চ | 🔴 উচ্চ | আগে প্রস্তুতি |
| পেমেন্ট জালিয়াতি | নিম্ন | উচ্চ | 🟠 মাঝারি | অ্যাকাউন্ট্যান্ট যাচাই |
| ডেটা ফাঁস | নিম্ন | অতি উচ্চ | 🔴 উচ্চ | অ্যাক্সেস নিয়ন্ত্রণ |
| ঝাপসা স্ক্যান | উচ্চ | নিম্ন | 🟡 নিম্ন | মান মানদণ্ড |

> 🔴 উচ্চ = তাৎক্ষণিক মনোযোগ; 🟠 মাঝারি = নিয়ন্ত্রণ; 🟡 নিম্ন = পর্যবেক্ষণ।
