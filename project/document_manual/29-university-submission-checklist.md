# অধ্যায় ২৯: ইউনিভার্সিটি সাবমিশন চেকলিস্ট

## ২৯.১ উদ্দেশ্য
ইউনিভার্সিটিতে আবেদন জমার আগে সব ডকুমেন্ট ও তথ্য সম্পূর্ণ ও নির্ভুল কিনা চূড়ান্তভাবে নিশ্চিত করা।

## ২৯.২ প্রি-সাবমিশন চেকলিস্ট
**পরিচয় ও একাডেমিক**
- [ ] পাসপোর্ট (মেয়াদ পর্যাপ্ত)
- [ ] NID/জন্ম সনদ (ইংরেজি)
- [ ] সব একাডেমিক সার্টিফিকেট
- [ ] সব ট্রান্সক্রিপ্ট (সম্পূর্ণ)
- [ ] IELTS (মেয়াদ বৈধ, যদি প্রযোজ্য)

**একাডেমিক লেখা**
- [ ] SOP (রিভিউড, প্লেজিয়ারিজম-মুক্ত)
- [ ] রিকমেন্ডেশন লেটার (যদি প্রযোজ্য)
- [ ] রিসার্চ প্রপোজাল (PhD/প্রযোজ্য)

**আর্থিক**
- [ ] ব্যাংক স্টেটমেন্ট (৬ মাস)
- [ ] সলভেন্সি সার্টিফিকেট
- [ ] e-TIN/ট্যাক্স (স্পনsor)
- [ ] ট্রেড লাইসেন্স/এমপ্লয়মেন্ট (প্রযোজ্য)

**প্রক্রিয়া**
- [ ] প্রয়োজনীয় অনুবাদ সম্পন্ন
- [ ] নোটারি/অ্যাটেস্টেশন সম্পন্ন
- [ ] সব ফাইল স্ট্যান্ডার্ড নামে
- [ ] পোর্টাল ভেরিফায়েড
- [ ] আবেদন ফি (যদি থাকে) পরিশোধিত

**ইউনিভার্সিটি-নির্দিষ্ট**
- [ ] অতিরিক্ত ফর্ম/প্রয়োজন যাচাই
- [ ] ডেডলাইন নিশ্চিত
- [ ] নাম সর্বত্র পাসপোর্ট অনুযায়ী এক

## ২৯.৩ সাবমিশন ধাপ
1. সিনিয়র/ম্যানেজার দিয়ে চূড়ান্ত QA (অধ্যায় ৩০–৩১)।
2. ইউনিভার্সিটি পোর্টাল/ইমেইলে আবেদন জমা।
3. সাবমিশন কনফার্মেশন/রেফারেন্স সংরক্ষণ।
4. Progress Sheet-এ "Submitted" + তারিখ।
5. অফার লেটারের জন্য ফলো-আপ।

## ২৯.৪ Red Flags / সাধারণ রিজেকশন
- ⚠️ অসম্পূর্ণ ফাইল/অনুপস্থিত ডকুমেন্ট।
- ⚠️ ডেডলাইন মিস।
- ⚠️ নাম গরমিল।
- ⚠️ QA ছাড়া সাবমিশন।

## ২৯.৫ বেস্ট প্র্যাকটিস
- ✅ কখনো QA ছাড়া সাবমিট নয়।
- ✅ ডেডলাইনের অন্তত ২-৩ দিন আগে প্রস্তুত।
- ✅ সাবমিশন কনফার্মেশন সবসময় সংরক্ষণ।
